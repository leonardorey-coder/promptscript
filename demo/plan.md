# Goal
Crear una landing page profesional de gatitos con un carrusel.

# Constraints
- Escribir solo en public/**
- No tocar src/** ni package.json
- No ejecutar comandos del sistema (RUN_CMD) en este demo
- Terminar con un REPORT que incluya filesChanged

# Steps
1. Crear public/index.html con: header, hero, sección de carrusel, CTA y footer.
2. Crear public/styles.css con diseño moderno y responsive.
3. Asegurar accesibilidad básica (aria-labels en botones del carrusel).
4. Reportar archivos modificados y sugerir siguientes pasos.
